# RestFramework

Requirements:
+ Boost 1.68
+ LibPQ 12 
+ GTest (for unit-tests) 

Build instructions:
1. Open root CMakeLists.txt
2. Specify paths: 
+ Boost_include_path
+ Boost_lib_path
+ libpq_include_path
+ libpq_lib_path
+ gtest_include
+ gtest_lib_path
3. Uncomment "set" instuctions 
4. Run CMake 
5. Build project
